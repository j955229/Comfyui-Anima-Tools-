# Anima Tools Wildcards

These files are plain one-line-per-choice wildcard lists for ComfyUI Impact Pack.

Files:

- `anima_tools/composition.txt`
- `anima_tools/expression.txt`
- `anima_tools/lighting.txt`

Typical wildcard names:

```text
__anima_tools/composition__
__anima_tools/expression__
__anima_tools/lighting__
```

If Impact Pack does not scan this custom node folder, copy the `anima_tools` folder into the Impact Pack wildcard directory.
